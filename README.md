# 🐾 Project Goal – Sistema de Gerenciamento de Petshop

## 📖 Sobre o Projeto

O **Project Goal** foi um desafio prático desenvolvido durante o bootcamp **Generation Brasil**, com duração de aproximadamente **4 horas**.

O objetivo foi desenvolver um sistema de gerenciamento para um **Petshop**, aplicando os principais conceitos de Programação Orientada a Objetos (POO) em Java e implementando operações de **CRUD (Create, Read, Update e Delete)**.

O projeto simula o gerenciamento de serviços ou registros do petshop, permitindo cadastrar, consultar, atualizar e excluir informações de forma organizada e eficiente.

---

## 🚀 Funcionalidades

- ✅ Cadastro de registros
- 📋 Listagem de registros
- 🔍 Consulta por identificação
- ✏️ Atualização de informações
- ❌ Exclusão de registros
- 🖥️ Menu interativo via console

## ▶️ Como Executar

### Pré-requisitos

- Java JDK 21 ou superior
- Eclipse IDE (ou outra IDE Java)

### Passos

1. Clone o repositório

```bash
git clone https://github.com/Joao-Pedro-Duo/projeto_final_bloco_01.git
```

2. Abra o projeto na IDE.

3. Execute a classe principal (`Menu.java` ou equivalente).

4. Utilize o menu para navegar pelas funcionalidades.
